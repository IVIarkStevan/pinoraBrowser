# Icons for Hello World Extension

This folder should contain PNG icon files:

- icon-16.png (16x16 pixels)
- icon-48.png (48x48 pixels)
- icon-128.png (128x128 pixels)

Create these images or download them from a free icon source and place them here.

You can use online tools to create simple icons:
1. Visit https://www.favicon-generator.org/
2. Create a simple icon (16x16, 48x48, 128x128)
3. Save the PNG files to this directory

Example: A simple gradient icon with "HW" text would work well for this extension.
